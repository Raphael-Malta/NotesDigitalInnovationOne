Stream API:
Manipulação de coleções com o paradgima funcional de paralela
Imutável - Não altera a coleção origem, sempre cria uma nova coleção
Principais funcionalidades
- Mapping - Retornada uma coleção com mesmo tamanho da origem com os elementos alterados
- Filtering - Retorna uma coleção igual ou menor que a coleção de origem, com elementos intactos
- ForEach - Executa uma determinada lógica para cada elemento, retornando nada.
- Peek - Executa uma determinada lógica para cada elemento, retornando a própria coleção
- Counting - Retorna uma inteiro que representa a contagem de elementos
- Grouping - Retorna uma coleção agrupada de acordo com a regra definida.
