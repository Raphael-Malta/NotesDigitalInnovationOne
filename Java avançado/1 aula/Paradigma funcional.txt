Paradigma Funcional no Java:
Programação funcional é o processo de construir software através de composição de funções puras, evitando
compartilhamento de estados, dados mutáveis e efeitos colaterais.
É declarativa ao invés de imperátiva, essa é uma definição de Eric Elliott.

Paradigma Imperativo:
E aquele que expressa o código através de comandos ao computador, nele é possível ter controle de estado dos objetos.
As instruções dele são executadas instantaneas.

Paradigma Funcional: 
Damos uma regra, uma declaração de como queremos que o programa se comporte.
